# AGENTS.md

このリポジトリで作業するエージェント向けのメモです。

## 概要

- 公開ドキュメント用の静的サイトです。
- ビルド工程や依存パッケージはありません。
- 既存の HTML/CSS/JS の素朴な構成を保ってください。

## 禁止事項

- 公開できない機密事項や個人情報はコードに一切含めてはいけない

## 作業ルール

- 変更は必要なファイルに限定してください。
- 既存のデザイン、余白、色、文体に合わせてください。
- ルートの `index.html` は全体入口、`quick-memo/` 以下は「3秒メモ」専用です。
- `quick-memo/site.js` は日英ページの共通ナビゲーションを生成します。リンク階層を変える場合は動作確認してください。
- 法務文書を編集する場合は、本文、HTML 版、日付、日英の対応関係を確認してください。

## プロジェクト追加

新しいアプリやプロジェクトを追加する場合の基本手順は、`README.md` の「プロジェクト追加手順」を参照してください。

プロジェクト固有のナビゲーション JS を追加する場合は、他プロジェクトのリンク構造に影響しないように分離してください。
