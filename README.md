# docs

公開用ドキュメントをまとめる静的サイトです。

現在は、iOS アプリ「3秒メモ」のドキュメントを配置しています。

## 構成

- `index.html`: 公開ドキュメント全体の入口
- `site.css`: 共通スタイル
- `assets/`: 共通画像など
- `quick-memo/`: 3秒メモ用ドキュメント
  - `ja/`: 日本語ページ
  - `en/`: 英語ページ
  - `site.js`: 言語切り替えとナビゲーション生成

## 更新方法

このリポジトリはビルドなしの静的 HTML/CSS/JS 構成です。

ローカル確認は、`index.html` をブラウザで確認。

## プロジェクト追加手順

新しいアプリやプロジェクトの公開ドキュメントを追加する場合は、以下の流れで作業します。

1. ルート直下にプロジェクト用ディレクトリを作成する
   - 例: `new-app/`
2. 必要に応じて言語別ディレクトリを作成する
   - 例: `new-app/ja/`, `new-app/en/`
3. 各言語の入口ページと文書ページを追加する
   - 例: `index.html`, `privacy.html`, `terms.html`
4. ルートの `index.html` にプロジェクトへのリンクを追加する
5. 共通スタイルで足りる場合は `site.css` を再利用し、必要な差分だけ追加する
6. 追加したページのリンク、言語切り替え、相対パスを確認する

既存の `quick-memo/` を雛形として使うと追加しやすいです。

## メモ

- `privacy.md` / `terms.md` は本文管理用、対応する `.html` は公開用ページです。
- 日本語と英語のページを更新する場合は、両言語で内容差分が意図したものか確認してください。
- 法務文書の内容を変更する場合は、公開前に文言と日付を確認してください。
